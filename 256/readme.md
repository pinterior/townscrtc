FA
--

 - screen
   - 640 x 480 8bpp
   - FA0 = 0x4000
   - FA1 = 0x8000

A red to blue gradation from top-left corner to bottom-right direction will be shown.

ZV
--

 - screen
   - 640 x 96, 640 x 30 8bpp
   - ZV0 = 0x4 (5x zoom)
   - ZV1 = 0xf (16x zoom)

Red-blue checkerbord pattern (8x6) will be shown.

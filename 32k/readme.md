FA
--

 - screen
   - 256 x 240 16bpp (2x zoom)
   - FA0 = 0x4000
   - FA1 = 0x8000

A red-green-blue-red gradation from top-left corner to bottom-right direction will be shown.

ZV
--

 - screen
   - 512 x 96, 512 x 30 16bpp
   - ZV0 = 0x4 (5x zoom)
   - ZV1 = 0xf (16x zoom)

Red-blue checkerbord pattern (6.4x6) will be shown.
